Ontario METAR Flight Conditions Project

This page hosts live-updating METAR data collected from 15 Ontario airports every 10 minutes.

Project Description

Data is collected via the CheckWX API and processed into categories (LIFR, IFR, MVFR, VFR). Over time, this dataset will allow analysis of seasonal, regional, and climatological flight conditions across Ontario.
